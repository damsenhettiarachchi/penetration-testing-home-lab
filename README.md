# Penetration Testing Home Lab

## Project Overview

This project demonstrates the creation of a beginner-friendly cybersecurity home lab using Kali Linux and Metasploitable 2 in a virtualized environment.

The goal was to gain hands-on experience with penetration testing fundamentals, network reconnaissance, and vulnerability assessment in a safe and controlled environment.

## Objectives

- Build a cybersecurity lab environment
- Configure communication between virtual machines
- Practice network reconnaissance
- Learn vulnerability assessment techniques
- Gain practical experience with cybersecurity tools

## Tools Used

- Kali Linux
- Metasploitable 2
- Nmap
- VirtualBox

## Activities Performed

1. Installed and configured Kali Linux
2. Installed Metasploitable 2
3. Configured virtual machine networking
4. Verified connectivity between machines
5. Identified target IP address
6. Performed network scanning using Nmap
